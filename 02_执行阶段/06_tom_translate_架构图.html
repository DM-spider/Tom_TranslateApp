服务器层
Backend Services FastAPI · Python 3.11+ · Uvicorn · pydantic-settings
packages/backend/
① 配置层 (config.py)
• pydantic-settings 从 .env 读取全局配置
• 包含 API Keys / Redis URL / CORS Origins / Rate Limits / Debug Mode
• @lru_cache 全局单例
② 中间件层 (middleware.py)
• CORS 跨域中间件（允许 Vercel 域 + chrome-extension://* origins）
• Rate limiting 中间件（滑动窗口算法，per IP，默认 30 requests/min，返回 429）
③ 路由层 (routers/translate.py, prefix /api/v1)
• POST /api/v1/translate — 核心翻译接口，接收 JSON，返回译文数组
• GET /api/v1/languages — 返回支持的语言列表
• GET /health — 健康检查
④ 服务调度层 (services/translator.py — TranslatorService)
Processes → 1 接收请求(多段文本)→ 2 Batch check Redis cache (get_batch)→ 3 找出未命中文本
requests → 4 调翻译引擎处理→ 5 新结果写 Redis (set, TTL 7 days)→ 6 Merge cached + 新翻译结果, 合并返回
⑤ 引擎层 (services/engines/)
BaseEngine: base.py — 抽象基类，定义 translate() 接口规范
DeepSeekEngine
deepseek.py：通过 OpenAI SDK 调用，兼容协议
GeminiEngine
gemini.py：通过 google-genai SDK 调用
BaiduEngine 待接入
baidu.py：httpx 异步调用，暂未启用
